This project is a Traffic Light Control System implemented in Python.
It simulates the behavior of traffic lights by managing multiple lanes and switching signals based on a predefined time for each lane.

Controls multiple traffic lanes

Automatic switching between lanes after a fixed green time

ðŸ”¹ How It Works

Each lane gets a green signal for a specific duration

When the time is over, the system switches to the next lane

After the last lane, it loops back to the first lane

Timing is handled using system time
